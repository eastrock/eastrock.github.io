Hexo尝试初步总结

准备工作

-Git

-Node.js

-Github账户

小提示

按照提示正确安装上述软件即可~如果不明白可以直接一直下一步傻瓜式安装

详细过程

过程因为邻近期末，不做赘述，在考试结束后会继续更新啦~

搭建心得

作为一个临近期末的大学生，理应把自己投入紧张的复习和预习工作中，但是按捺不住心中的冲动，想尝试一下这个效果，也为自己后续的学习做一点总结~
搭建过程因为有学长之前做过，所以过程整体还算顺利，但是也有一些小的细节，比如在添加SSH的时候可以用GITHUB的官方客户端登录一下就自动生成、用Everthing可以快速定位文件位置，在hexo的3.0版本之后

type: git

同时在yml文件中，代码：之后一定要有空格，最初的时候遇到好几次的报错，后来才慢慢习惯这个语法~

主题更换

Hexo提供了很多的博客主题供大家更换，详情可以登录Hexo官网点击Themes选项，或者网友提供的主题
